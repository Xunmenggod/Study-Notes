# Linux Commands Record

## Normal command
```bash
ls # list all the file
pwd # print work directory path
mkdir <dir name> # make directory
touch <file name> # create a new file
ls | grep # grep filter information 
cat # show file content
tar zxf <tar.gz file name> -C <path of the extraction>
unzip/zip # to unzip or zip to a .zip file
&& # to connect two command and execute in serial
```
